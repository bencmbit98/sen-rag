# PDF RAG App (Streamlit + LangChain)

A minimal Retrieval-Augmented Generation (RAG) application that allows users to ask questions over PDF documents.

## Features
- PDF ingestion
- Vector search with FAISS
- OpenAI LLM for answers
- Streamlit UI
- Deployable on Streamlit Community Cloud

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
